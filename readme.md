## Laravel-Ajax-DataTables

  DataTable Ajax Implementation using Laravel Framework

  Tools Used

  -Laravel 
  -DataTables
  -jQuery


##Instruction

1. Clone the Repo
2. Upload the airplane_model.sql file to your phpMyAdmin or any similar tool
3. Run the app using artisan php artisan serve
